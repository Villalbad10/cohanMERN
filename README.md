## 🚀 Get Started

Necesitas estar usando:

- [Git](https://git-scm.com/downloads)
- [Nodejs v18 or higher](https://nodejs.org/download/)

Clone the repository & install dependencies:

```bash
git clone https://github.com/Villalbad10/cohanMERN.git
cd cohanMERN
npm install
npm start
```
